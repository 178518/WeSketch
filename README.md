# WeSketch 

**WeSketch** 是一个基于 Sketch 的一个插件集合，是微信设计团队使用 Sketch 设计和还原过程中对 Sketch 一些些不好用的小改进。

[下载解压后双击wesketch.sketchplugin文件安装](https://github.com/weixin/wesketch/archive/master.zip)
